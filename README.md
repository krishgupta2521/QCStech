# NexGen IT Landing Page

A Vite + React single-page site showcasing NexGen IT services, solutions, and contact workflows.

## Quick Start

- Install dependencies: `npm install`
- Start dev server: `npm run dev`
- Build for production: `npm run build`
- Preview build: `npm run preview`

## Notes

- UI built with Tailwind CSS and Lucide icons.
- Form actions are client-side only (no backend). If you want a real submission endpoint, add it to the form handler in `src/App.jsx`.
